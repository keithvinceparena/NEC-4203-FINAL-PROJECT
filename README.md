# S.A.F.E.R SYSTEM  
## Smart, Adaptive, and Flexi-Elevating Roadbreaker System

The *S.A.F.E.R System* is a smart road safety prototype that detects vehicle speed, adapts to day and night conditions, elevates a roadbreaker for overspeeding vehicles, displays real-time information, and sends wireless alerts to a phone.

This project uses:
- *Arduino Uno* for sensing, decision-making, display, and actuation
- *ESP32* for wireless communication and phone alerts
- *IR sensors* for vehicle speed detection
- *Grove Light Sensor* for day/night adaptive control
- *Servo motor* for elevating the roadbreaker or the bump
- *LCD with I2C* for system feedback
